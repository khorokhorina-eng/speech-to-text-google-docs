# Chrome Web Store Copy

## Product Name

Speech to Text Google Docs

## Short Description

Use Speech to Text Google Docs for browser-based voice to text in Google Docs, with direct dictation and simple controls.

## Detailed Description

Speech to Text Google Docs helps you dictate directly into Google Docs while working in the browser.

Use it to:

- dictate notes, drafts, and outlines without typing
- dictate directly into Google Docs
- use simple voice commands for punctuation while you speak
- use voice-friendly punctuation commands such as `comma`, `period`, and `new paragraph`

How it works:

- open a Google Docs document
- start dictation from the extension
- speak and watch text appear in your document
- stop when you are done

Account and billing:

- includes 15 free sessions
- continued use requires a paid plan
- Google sign-in is required before checkout
- billing is handled through Stripe

Plans:

- Monthly: unlimited dictation
- Annual: unlimited dictation

This extension is designed for Google Docs workflows and is intended for writing, note-taking, drafting, and general speech-to-text input in the browser.

## Privacy Disclosure

This extension processes microphone audio, dictated text, account, billing, and trial-session data through `voicetext.world` to support browser speech recognition workflows, free trial limits, Google sign-in, subscription status, and Stripe checkout. Support contact: `hello@voicetext.world`.
